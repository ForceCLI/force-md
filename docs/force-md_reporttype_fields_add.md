## force-md reporttype fields add

Add field to report type

### Synopsis

Add field to report type

```
force-md reporttype fields add -s Section -t Table -f Field [filename]...
```

### Options

```
  -f, --field string     field name
  -h, --help             help for add
  -s, --section string   section
  -t, --table string     table name
```
