## force-md profile object-permissions add

Add object permissions

```
force-md profile object-permissions add -o SObject [flags] [filename]...
```

  -e, --edit            allow edit
  -r, --read            allow read
  -m, --modify-all      allow modify all
  -v, --view-all        allow view all
