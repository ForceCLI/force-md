## force-md

force-md manipulate Salesforce metadata

```
force-md
```

### Options

```
      --convert-xml-entities   convert numeric xml entities to character entities (default true)
  -h, --help                   help for force-md
      --silent                 show errors only
      --verbose                show debugging output
```
