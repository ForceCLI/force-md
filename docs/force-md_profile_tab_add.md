## force-md profile tab add

Add tab visibility

```
force-md profile tab add -t TabName [flags] [filename]...
```

### Options

```
  -h, --help                    help for add
  -t, --tab string              tab name
  -v, --visibility visibility   tab visibility; can be 'DefaultOn', 'DefaultOff', or 'Hidden' (default defaulton)
```
